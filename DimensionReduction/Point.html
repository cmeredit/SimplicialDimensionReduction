  •  package DimensionReduction

    Provides utility objects for dimension reduction.

    Provides utility objects for dimension reduction.

    Overview

    The subpackages of this package are DBSCAN and Delaunay. The former package provides an implementation of the DBSCAN algorithm, while the latter package provides algorithms that produce the Delaunay simplicialization and convex hull of a set of points. The Delaunay package also has a Linear Algebra helper object, Delaunay.LinearUtil

  •  package DBSCAN

    Provides objects and traits for implementing the DBSCAN algorithm.

    Provides objects and traits for implementing the DBSCAN algorithm.

    Overview

    The main object to use is DimensionReduction.DBSCAN.DBSCANUtil, e.g.,

    val epsilon: Double = 0.1
val minPts: Int = 3
val points: Vector[Point] = Vector(Point(Vector(0.0, 0.0, 0.0)), Point(Vector(0.0, 0.0, 1.0)))
val clusteredPoints: Vector[DBPoint] = DBSCANUtil.cluster(epsilon, minPts)(points)

    If you intend to cluster multiple sets of points according to the same parameters, you can fill just the first parameter list of DBSCANUtil.cluster:

    val epsilon: Double = ???
val minPts: Int = ???
val clusterer: Vector[Point] => Vector[DBPoint] = DBSCANUtil.cluster(epsilon, minPts)

val firstCollectionOfPoints: Vector[Point] = ???
val secondCollectionOfPoints: Vector[Point] = ???

val firstClusteredPoints: Vector[DBPoint] = clusterer(firstCollectionOfPoints)
val secondClusteredPoints: Vector[DBPoint] = clusterer(secondCollectionOfPoints)

    Overview

    The main objects in this package are DimensionReduction.Delaunay.DelaunayUtil and DimensionReduction.Delaunay.QuickHullUtil. These objects provide methods for obtaining the Delaunay triangulation and convex hull of a set of points, respectively.

Point

case class Point(coordinates: Vector[Double]) extends Product with Serializable

Represents a point in space of arbitrary dimension.

The primary purpose of this class is to augment Vector[Double] with helper functions and attributes. Operations are generally only well defined for points of the same dimension. Most functions in this class therefore return Option[Point]

coordinates

The coordinates of the point.
