  •  package DimensionReduction

    Provides utility objects for dimension reduction.

    Provides utility objects for dimension reduction.

    Overview

    The subpackages of this package are DBSCAN and Delaunay. The former package provides an implementation of the DBSCAN algorithm, while the latter package provides algorithms that produce the Delaunay simplicialization and convex hull of a set of points. The Delaunay package also has a Linear Algebra helper object, LinearUtil

  •  package DBSCAN

    Provides objects and traits for implementing the DBSCAN algorithm.

    Provides objects and traits for implementing the DBSCAN algorithm.

    Overview

    The main object to use is DimensionReduction.DBSCAN.DBSCANUtil, e.g.,

    val epsilon: Double = 0.1
val minPts: Int = 3
val points: Vector[Point] = Vector(Point(Vector(0.0, 0.0, 0.0)), Point(Vector(0.0, 0.0, 1.0)))
val clusteredPoints: Vector[DBPoint] = DBSCANUtil.cluster(epsilon, minPts)(points)

    If you intend to cluster multiple sets of points according to the same parameters, you can fill just the first parameter list of DBSCANUtil.cluster:

    val epsilon: Double = ???
val minPts: Int = ???
val clusterer: Vector[Point] => Vector[DBPoint] = DBSCANUtil.cluster(epsilon, minPts)

val firstCollectionOfPoints: Vector[Point] = ???
val secondCollectionOfPoints: Vector[Point] = ???

val firstClusteredPoints: Vector[DBPoint] = clusterer(firstCollectionOfPoints)
val secondClusteredPoints: Vector[DBPoint] = clusterer(secondCollectionOfPoints)

  1.  object DelaunayUtil

    Provides functions to compute the Delaunay simplicialization of a collection of points of uniform dimension.

    Provides functions to compute the Delaunay simplicialization of a collection of points of uniform dimension.

    In two dimensions, the Delaunay simplicialization is the usual Delaunay triangulation. I.e., a triangulation of the points such that no circumcircle of any triangle contains another point in its interior.

    In three dimensions, the Delaunay simplicialization of a set of points is a collection of tetrahedra such that no circumsphere of any tetrahedron contains another point in its interior.

    In n dimensions, the Delaunay simplicialization of a set of points is a collection of n-simplices such that no circumhypersphere of any n-simplex contains another point in its interior.
